# Blackstone Elevators — Web

Modern marketing site for Blackstone Elevators (Chennai). Vite + React 19 + TypeScript + Tailwind + Framer Motion + MDX.

## Develop

```
npm install
npm run dev
```

Opens at http://localhost:5173.

## Build

```
npm run build
npm run preview
```

## Deploy

Configured for Vercel. Push to a git remote and import the project; `vercel.json` handles the rest (SPA rewrites, Vite framework preset).
